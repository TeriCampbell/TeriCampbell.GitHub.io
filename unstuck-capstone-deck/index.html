Why this problem is worth solving now

  • High school students juggle many deadlines. Overwhelm can escalate quickly once they feel behind.
  • Unstuck addresses the paralysis around where to start with one clear next action.
  • Plans include resource links and context, so students can find the right support later.
  • Adult support links can trigger account setup help or counselor check-ins when needed.

Big picture

  • A problem worth solving: paralysis shows up before grades slip; counselors cannot scale 1:1 at 372:1
  • What is effective and feasible:
    • Structured intake and allowlisted resources, not open chat or RAG
    • Haiku for screening and task prefill; Sonnet for plans (affordability vs depth)
    • SAT and resource rules enforced in code, not prompt-only
    • Pilot-ready now with anonymous sessions; OAuth and email check-in deferred
  • Measuring success: first plan saved, return visits (visitor_id), check-in outcomes, resource link clicks (events), and counselor-contact signal from check-in

Ships today (student path)

  • Situation: structured intake and screened optional text
  • Tasks and Clarify: deadlines and context for the plan
  • Plan: one next action and allowlisted resources
  • Dashboard: return hub and focus task
  • Toolkit: resources tied to the plan
  • Check-in: weekly progress loop
  • Safety: distress screening on free-text fields

Ships next

  • Counselor onboarding: verified school accounts and rollout workflow
  • Parent and adult help: scoped notifications for resource setup
  • School configuration: institution-specific resources and compliance packet
  • Identity and reach: OAuth login and tokenized email check-in links
  • Institutional buyer path: counselor dashboard beyond demo aggregates (see slide 8)
